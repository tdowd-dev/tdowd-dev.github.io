About Me

Hello, my name is Trevor and I am an aspiring software & web developer. I studied at Western New England University for 4 years, graduating Summa Cum Laude with a B.S. in Computer Science. I am a very detail-oriented person who loves to learn.

My coding journey began in high school when a simple HTML class sparked what would become a burning passion. Going into college I was moved away from web development, but found the challenge of solving problems in Python and Java to be very rewarding. In the latter half of my college experience, I enjoyed having a heavy focus on working within Agile teams to develop and contribute to large-scale FOSS projects covering a wide range of technologies, including Docker and Node.js.

Now, I am currently working at LLumin, Inc., which has lead me back to the world of web development where my journey began, and I am excited to see where it will take me next!

A FOSS project under the LibreFoodPantry community. The project is a web application with a microservices architecture to be used on a single system for food pantry employees to keep track of inventory and guest visits. I worked as a developer, and eventually Scrum master, of a team among other Scrum teams to make contributions to the project, with our biggest focus being the implementation of development tools such as development containers and hot reloading. In addition to implementing new features, we also created and maintained documentation, debugged manual and automatic testing, worked with a CI/CD pipeline, and practiced Agile methodologies such as weekly sprint reviews, retrospectives, and peer feedback. We also gained first-hand experience with the communication between frontend and backend systems via an Open Rest API.

ROI Calculator

Technologies used: HubSpot, WordPress, CSS, HTML, JavaScript, Figma.

Try It Yourself!

A form created for LLumin, Inc. using HubSpot's forms, workflows, emails, and properties features. The form can be opened either through following a link (see button above) or soon to be embedded on a page in LLumin's website with additional CSS styling using WordPress. After filling out the form, users can enter their contact info to receive an ROI report by email based on their input.

To create the form, I first created a mockup design in Figma, then designed a simple implementation in HTML and JavaScript, before testing with various styles of forms in HubSpot. After some collaboration with the product owners, I was able to come up with a set of inputs, outputs, and calculations that aligned with their expectations. I received positive feedback saying that the form looks great and is likely to increase the quantity and length of user interactions on the website.

Database Design Project

Technologies used: SQL, MySQLWorkbench, E-R Diagrams

Unfortunately, I no longer have access to the original SQL database. However, the full documentation of each iteration of the design can be downloaded below:

Download Docs

A class project database created iteratively from scratch using MySQLWorkbench. Before the final implementation in SQL, I collaborated with another student to create conceptual, logical, and physical designs. For these designs we made use of tables, and E-R diagrams to represent relations, relationships, and attributes. Our designs and implementation also included stored procedures, functions, and views. We considered relationship types, normalization, redundancy, and use cases. Our topic of choice for the database was to hold information about Mario Kart games.

A command line script written in python and compiled into an executable for ease of use. The script takes a file as input and then copies it into a specific series of folders for game modding purposes. It makes use of various libraries to perform file manipulation.
